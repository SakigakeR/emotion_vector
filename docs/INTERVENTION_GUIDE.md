# 情绪干预参数建议表

本文档提供不同模型架构下的情绪干预参数建议，包括推荐的干预层和干预强度。

## 参数说明

| 参数 | 说明 |
|------|------|
| **模型名** | HuggingFace 模型标识符 |
| **参数量** | 模型总参数量 |
| **层数** | Transformer layers |
| **隐藏维度** | 模型隐藏层维度（向量维度） |
| **推荐干预层** | 建议的干预层范围 |
| **推荐干预强度** | 建议的强度系数范围 |

## 实验调整指南

1. **从推荐值开始**: 使用表中推荐的中间值作为起点
2. **逐步调整**: 通过二分法逼近合适干预强度
3. **观察输出**: 注意输出是否自然、是否出现重复或崩溃
4. **记录效果**: 记录不同参数下的效果，建立自己的经验表

## 注意事项

- 干预强度与模型温度（temperature）参数相互影响
- 过高的强度可能导致输出质量下降
- 建议先在小样本上测试，再应用到正式场景
